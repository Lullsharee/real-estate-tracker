# 不動産物件情報 データソース調査結果

## 推奨: 国土交通省 不動産取引価格情報API

### URL
https://www.land.mlit.go.jp/webland/api.html

### 利用規約
- **利用可能**: Public Data License (Version 1.0) に基づき自由に利用可能
- **商用利用**: 可能
- **出典表示**: 必要（例: "出典: 国土交通省ウェブサイト"）
- **データ編集**: 可能（編集した旨を明記）

### 提供データ
- 不動産取引価格情報（実際の取引価格）
- 地域: 全国（東京・埼玉・神奈川含む）
- 物件種別: 宅地、中古マンション、農地、林地
- 取引時期、面積、価格、坪単価など

### API仕様
- REST API
- JSON形式
- 認証不要
- 無料

### エンドポイント例
```
GET https://www.land.mlit.go.jp/webland/api/TradeListSearch
パラメータ:
- from: 取引時期（開始）
- to: 取引時期（終了）
- area: 都道府県コード（13:東京、11:埼玉、14:神奈川）
- city: 市区町村コード
```

## 結論

**国土交通省の不動産取引価格情報APIを使用することを強く推奨**

### 理由
1. ✅ 公式API - 安定性が高い
2. ✅ 利用規約が明確 - 商用利用可能
3. ✅ 無料・認証不要
4. ✅ 法的リスクなし
5. ✅ 実際の取引価格データ（信頼性が高い）

### 制限事項
- リアルタイムの募集物件情報ではなく、過去の取引実績データ
- 更新頻度: 四半期ごと
- 個別物件の詳細情報（間取り、写真等）は含まれない

### 代替案
もし現在募集中の物件情報が必要な場合:
1. 不動産業者向けAPI（有料）の利用を検討
2. 公開されている不動産オープンデータの活用
3. 手動でのデータ収集・入力
