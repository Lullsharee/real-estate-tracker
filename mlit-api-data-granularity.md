# 国土交通省 不動産取引価格情報API データ粒度

## データの粒度

### 地理的粒度
1. **都道府県レベル**: 可能
2. **市区町村レベル**: 可能
3. **地区詳細（町丁目）**: **含まれる**
4. **最寄り駅**: **含まれる**

## 粒度の詳細

### ✅ 利用可能な粒度
- **都道府県**: 東京都、埼玉県、神奈川県
- **市区町村**: 千代田区、横浜市、さいたま市など
- **地区（町丁目）**: 神田神保町、渋谷、横浜駅西口など
- **最寄り駅**: 駅名 + 徒歩時間

### 📊 分析可能な内容
1. **地区別の坪単価推移**
   - 例: 渋谷区の各町丁目ごとの平均坪単価
   
2. **駅別の価格分析**
   - 例: 渋谷駅徒歩5分圏内の平均価格
   
3. **市区町村別の比較**
   - 例: 千代田区 vs 港区 vs 渋谷区

4. **時系列分析**
   - 四半期ごとの価格推移
   - 平均値・中央値の算出

## 制限事項

### ❌ 含まれないデータ
- 物件の住所（番地まで）
- 物件の写真
- 現在募集中の物件情報
- リアルタイムデータ（四半期ごとの更新）
- 個別物件の識別情報

### 更新頻度
- **四半期ごと**（3ヶ月に1回）
- 最新データは約1-2ヶ月遅れ

## 結論

**地区（町丁目）レベルと最寄り駅情報が含まれるため、十分に詳細な分析が可能です。**

### できること
✅ 渋谷区神南の坪単価推移
✅ 渋谷駅徒歩5分圏内の平均価格
✅ 東京23区の地区別価格マップ
✅ 駅距離別の価格分析

### できないこと
❌ リアルタイムの募集物件情報
❌ 物件の写真や詳細な設備情報
❌ 番地レベルの正確な位置情報
